# Fish Weight Prediction using Linear Regression

## Overview
This project predicts the weight of a fish using the Linear Regression algorithm based on its physical measurements.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
- Fish Market Dataset
- Features: Species, Length1, Length2, Length3, Height, Width
- Target: Weight

## How to Run
```bash
pip install pandas matplotlib scikit-learn
python regression.py
```

## Output
- Fish weight prediction
- MAE
- MSE
- RMSE
- R² Score
- Actual vs Predicted graph
